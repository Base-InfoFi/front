# Front - Yapper Leaderboard

Pre-TGE Arena 리더보드 프론트엔드 애플리케이션

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3001](http://localhost:3001)을 열어 확인하세요.

### 환경 변수

프로젝트에 `.env.local` 파일이 이미 생성되어 있습니다. 필요시 수정하세요:

```
NEXT_PUBLIC_BACKEND_URL=http://localhost:3000
```

> 참고: `.env.example` 파일을 참고하여 환경 변수를 설정할 수 있습니다.

## 빌드

```bash
npm run build
npm start
```
